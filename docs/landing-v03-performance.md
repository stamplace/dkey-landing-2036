# DKey Landing v0.3 — Stable Bright Product

## Decision
הדף הקודם היה כבד וקופצני מדי. v0.3 עובר לגישה יציבה:
מוצר בהיר, תנועה מינימלית, קבצים מפוצלים, בלי קנבס ובלי אנימציות כבדות.

## Removed
- canvas background
- phone floating animation
- spinning halo
- heavy blur/backdrop effects
- single giant HTML file

## Kept
- phone as main product
- keyboard action layers
- tap once / use everywhere message
- privacy and human approval
- controlled typing demo

## Performance rule
הדף צריך להרגיש חי בגלל המוצר, לא בגלל שהעמוד זז.
