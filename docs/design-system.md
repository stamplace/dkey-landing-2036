# DKey Design System

## שפה
עתידית, נקייה, חדה, מקצועית, 2036.

## צבעים
- ירוק DKey: פעולה, חיים, אישור
- שחור/כחול כהה: עומק, מערכת, רצינות
- לבן זכוכיתי: מוצר, ניקיון, אמון
- סגול עדין: יצירה ומצבים

## עקרונות UI
- בלי טקסטים צפופים
- כפתורים קצרים
- אין גלישת אותיות
- כל שכבה צריכה להיות מובנת בפני עצמה
- הטלפון הוא המוצר המרכזי
- התנועה משרתת הבנה, לא קישוט

## אסור
- טקסט שבור
- אימוג׳י לא קריא
- עומס מילים
- אפקטים שמסתירים את המקלדת
- תנועה שמקפיצה את הטלפון
