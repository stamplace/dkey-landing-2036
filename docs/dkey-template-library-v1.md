# DKey Template Library v1

## Status
Active demo library.

## Principle
לא אינסוף תבניות ולא כאוס.
מעט קטגוריות חזקות שמייצגות את דרך העבודה של המשתמש בתחומי החיים המרכזיים.

## Categories
- DEV — בנייה, בדיקה, שינוי קטן, אישור
- DKEY — שיטת עבודה אישית מעל כל כלי
- SONG — מוזיקה, מילים, הוק, סגנון
- BUSINESS — הצעה, ערך, לקוח, סגירה
- STAM — סת״ם, בדיקה תמה, דיוק, כבוד
- PRODUCT — DKey / מד קסת / מוצר אמיתי
- SAFE — פרטיות, אישור, בלי שליחה אוטומטית
- MESSAGE — הודעה אנושית, קצרה, מדויקת

## Design rule
כל תוצאה חייבת להיות:
קצרה, שימושית, מכובדת, מדויקת, ומוכנה להעתקה ידנית.

## Forbidden
- תהליך אפור
- טקסט אינסופי
- הבטחות לא מבוססות
- פעולה אוטומטית בלי אישור
- ניסוחים מנופחים
