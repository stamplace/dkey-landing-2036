# DKey Final Landing Content v1

## Core sentence
DKey is the action layer above the keyboard:
write naturally, choose a mode, pick an action, get a prepared result, approve manually.

## Hebrew core
DKey היא שכבת פעולה מעל המקלדת:
כותבים טבעי, בוחרים מצב, בוחרים פעולה, מקבלים תוצאה מוכנה, ומאשרים ידנית.

## What DKey is
- שכבת עבודה אישית מעל כל שדה טקסט.
- דרך קבועה לעבוד מול כל מודל וכל כלי.
- מקלדת שמחזיקה מצבים, פעולות, תבניות ואישור.
- מוצר שמחזיר את האדם למרכז.

## What DKey is not
- לא מודל AI נוסף.
- לא תחליף ל־GPT / Gemini / Grok / Suno.
- לא agent שמריץ דברים לבד.
- לא מערכת ששולחת במקומך.
- לא קסם; שכבת פעולה מסודרת.

## Product flow
1. Write — האדם כותב טבעי.
2. Choose mode — DKey / DEV / SUNO / Create / Safe.
3. Pick action — פעולה מוגדרת, לא כאוס.
4. Review result — תוצאה מוכנה לשימוש.
5. Approve manually — האדם מחליט.

## Trust rules
- No auto-send.
- No auto-run.
- Human approval before action.
- Keep the user in control.
- Do not promise technical privacy claims that are not implemented and verified.

## Landing page goal
אדם שנכנס לדף צריך להבין:
זה לא עוד כלי AI. זו שכבת הפיקוד שלי מול כל כלי AI.

## Audience
- מפתח
- יוצר
- מוזיקאי
- בעל עסק
- אדם שעובד הרבה עם טקסט, רעיונות וכלי AI

## Tone
Clear. Calm. Strong. Product-level. Not hype. Not placeholder.
